# A Program to Swim Relays into 4 Videos
- To speed up the process of spliting a swim relay video into 4 videos (one for each athlete)
- Provides a variable buffer time on around the selecting split time to give some space on either side of the start and finish of each athlete
- Maintains all initial video quality and executes relativly quickly
- Uses Qt for UI and FFMPEG for video transcoding
- FFMPEG.exe is not included in the repo and will need to be added to the "ffmpeg" folder in order to transcode videos

- This program may not work for Mac users as the ffmpeg file location may need to be changed
